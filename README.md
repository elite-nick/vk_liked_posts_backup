# VK Liked Posts Backup

Python-скрипт для скачивания **всех лайкнутых постов** из указанной группы VK/ВКонтакте  
со **всеми вложениями**, включая **музыку**! Скрипт ориентирован на **архивацию контента** и работает **по токену пользователя**.

## Для чего я сделал этот скрипт?
Ранее у меня была любимая группа с музыкой, с которой я качал себе локально музыку очень давно. Потом мой жесткий диск сломался и вся скачанная музыка пропала... Я помимо скачивания еще и лайкал посты с музыкой, ну и получается отметки любимых песен в группе есть. Я не нашел в интернете такого решения, но решил его воплотить в реальность!

## Возможности скрипта

- Скачивание **только лайкнутых постов**
- Работа только **с одной конкретной группой** (указывается ее ID)
- Скачивание **всех вложений**:
  - Фото (максимальный размер)
  - Документы
  - Видео
  - Музыка (на музыку сделан упор, к ней добавляются ID3 теги и скачивается музыка корректно через HLS)
- **Прогресс выполнения** (в процентах)
- Возможность указать, **с какого поста начинать загрузку**

## Подготовка к настройке и запуску

- Установите зависимость mutagen (pip install requests mutagen)
- Скачайте FFmpeg.exe и оставьте его рядом со скриптом
- В начале файла main.py укажите следующее:
  - TOKEN = "ТВОЙ_ТОКЕН" (Получить его можно здесь - https://vkhost.github.io/; Выбрать vk.com)
  - GROUP_ID = -123456789 (ID группы ВК)
  - START_POST_ID = None (ID поста, с какого начинать поиск; None — начать с самого начала; 123456 — начать с конкретного поста)

## Запуск
После выполнения подготовки просто запустите скрипт (python main.py) и ждите когда скачаются все лайкнутые посты! Они будут скачаны в папке Download рядом со скриптом.
